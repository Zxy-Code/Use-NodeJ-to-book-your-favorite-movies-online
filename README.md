# 使用NodeJs来构建一个在网上预订电影票的网站
## 工具
- 后端：NodeJs
- 前端：Js Html Css
## 运行项目
```
node index.js
```
## 课程

	 	2.代码不够优美（项目原则是：用原生最直白的思路去实现功能，方便初学者很容易的看懂，代码优化后期统一做）
